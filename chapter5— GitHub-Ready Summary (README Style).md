## MatsuiOS Directed Adaptive Derivative (MDAD)

MDAD is an OS-driven derivative concept where the direction, scale, and mode
are generated by MatsuiOS rather than chosen manually.

### Key Properties
- Direction generation (OS-dir)
- Scale generation (OS Geometry)
- Automatic singularity avoidance (complex-step)
- High numerical stability

### Continuous Definition

$$
D_{\text{MDAD}} f(x) = \lim_{\varepsilon \to 0} \frac{ f(x + \varepsilon\,\ell_{\text{OS}}(x)\,\vec d_{\text{OS}}(x)) - f(x) }{ \varepsilon\,\ell_{\text{OS}}(x) }
$$

### Discrete Definition

$$
D_{\text{MDAD}}^{\text{disc}} f(x) = \frac{ f(x + \ell_{\text{OS}}(x)\,\vec d_{\text{OS}}(x)) - f(x) }{ \ell_{\text{OS}}(x) }
$$
